# Driver Drowsiness Detection System 🚗😴

This project detects driver drowsiness using computer vision.

## 🔥 Features
- Face Detection
- Eye Detection
- Drowsiness Alert
- Real-time Monitoring

## 🛠️ Technologies
- Python
- OpenCV

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python main.py

## 🎯 Output
- Detects face and eyes
- Alerts when driver is drowsy
